# Rasadhi Platform

Production cheminformatics platform for QSAR preprocessing, analysis, and modeling.

## Status

Alpha — under active development. Not yet ready for public use.

## Structure

This is a monorepo containing:

- **backend/** — FastAPI REST API ([README](backend/README.md))
- **frontend/** — React + TypeScript SPA ([README](frontend/README.md)) _(coming soon)_
- **docs/** — Architecture and deployment documentation

## Related projects

- [rasadhi_core](https://github.com/Papular108/rasadhi_core) — Pure scientific library consumed by this platform
- [qsar-preprocessing-tool](https://github.com/Papular108/qsar-preprocessing-tool) — Original Streamlit prototype
